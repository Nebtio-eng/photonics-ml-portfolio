# Photonics + ML Design Portfolio

This repository contains a structured portfolio of projects in
integrated photonics and physics-guided machine learning.

Each project is self-contained and follows a consistent workflow:
physics → simulation → ML surrogate → robustness analysis → design insight.

## 🧠 Design Philosophy
- Physics-first modeling
- ML used as a surrogate, not a black box
- Robustness evaluated at decision time
- Clean, reproducible project structure
