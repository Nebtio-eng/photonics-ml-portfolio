# Project 2 – Silicon Ring Resonator Simulation

## Objective
To simulate a side-coupled silicon microring resonator and extract:
- Transmission spectrum
- Free Spectral Range (FSR)
- Q-factor
- Group index (ng)
- Critical coupling behavior

## Data-Driven Extension
A structured dataset was generated by sweeping ring radius and coupling gap.
Extracted features include:
- Free Spectral Range (FSR)
- Q-factor
- Dip depth

This dataset enables ML regression and inverse photonic design.


## Tools
- MEEP (FDTD)
- Python (NumPy, SciPy, Matplotlib)
- VS Code + WSL

## Key Results
- Normalized resonance spectra obtained
- FSR increases as ring radius decreases
- Q-factor extracted from linewidth
- Group index estimated using frequency-domain FSR
- Critical coupling identified via gap sweep

## Files
- `ring_simulation.py` – main simulation
- `fsr_ng_analysis.py` – group index extraction
- `coupling_sweep.py` – coupling study

This project demonstrates a full photonic design workflow used in integrated photonics research and industry.
